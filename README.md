## 📌 Sobre o Projeto

O **Painel de Tabuadas** é uma aplicação web interativa desenvolvida com **HTML, CSS e JavaScript puro**, que gera automaticamente as tabuadas do número **1 até 100**, exibindo cada uma em formato de cards organizados.

O projeto foi criado com foco educacional para praticar:

- Lógica de programação
- Estruturas de repetição
- Manipulação do DOM
- Organização visual com CSS
- Interatividade com JavaScript

---

## 🎯 Funcionalidades

✔ Geração automática das tabuadas de 1 até 100

## ⚙️ Como Funciona

O JavaScript percorre os números de 1 até 100 utilizando um loop:

```javascript
for (let numero = 1; numero <= 100; numero++)
```

Para cada número, ele gera sua tabuada de 1 até 10:

```javascript
for (let i = 1; i <= 10; i++)
```

E cria dinamicamente um card com os resultados.
